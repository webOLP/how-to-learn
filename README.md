# Сайт "Научиться учиться"
## Описание проекта
Сайт "Научиться учиться" призван помогать людям научиться правильно распределять время и силы при получении новой информации. Пользователь ознакомится с несколькими методами и принципами обучения, что благоприятно скажется на его эффективности.
В проекте использованы трансформации и анимации. Код проекта использует методологию БЭМ, как и его структура.
